# Ham Radio Notes

A topic-based GitHub Pages site for ham radio articles, notes, and experiments.

## Structure

- `cw/` Morse, QRQ, training
- `sota/` portable activation notes
- `antennas/` antenna articles
- `station/` equipment and setup
- `operating/` operating practices
- `experiments/` tests and comparisons

## Adding a new article

1. Pick the right folder
2. Create a new `.md` file
3. Add front matter
4. Link it from that folder's `index.md`
